Menunggu token baru muncul di Pump.fun…
Catatan: feed live ini berjalan langsung dari browser kamu via WebSocket ke Solana RPC (logsSubscribe ke program Pump.fun) — tidak ada server perantara. Default RPC sudah dipasang ke Helius sesuai yang kamu kasih. Karena ini file client-side, key tersebut akan terlihat siapa saja yang melihat source/network tab kalau web ini dipublikasikan — kalau mau publik, pakai key khusus dengan rate-limit dibatasi di dashboard Helius. Data harga & liquidity diambil dari API publik Dexscreener — token yang masih di bonding curve Pump.fun (belum migrasi ke Raydium) belum akan punya data ini, chart-nya pun belum tersedia di Dexscreener. Fitur Buy/Sell membangun transaksi lewat API publik PumpPortal (pihak ketiga, bukan afiliasi Anthropic) dan menandatanganinya lewat Phantom kamu sendiri — non-custodial. Selalu periksa detail transaksi di popup Phantom sebelum approve, dan coba nominal kecil dulu. Parsing event Pump.fun dibuat manual berdasar struktur program publiknya; jika Pump.fun mengubah programnya, parsing perlu di-update.
